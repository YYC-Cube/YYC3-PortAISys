### 1. ✅ OpenAI适配器（OpenAIAdapter）

**支持的模型**：
- GPT-4 Turbo
- GPT-4
- GPT-3.5 Turbo
- 等所有OpenAI模型

**核心功能**：
- ✅ 聊天补全（Chat Completions API）
- ✅ 文本补全（Completions API）
- ✅ 自动重试机制（指数退避）
- ✅ 完整的错误处理
- ✅ API密钥和Organization支持
- ✅ 自定义Base URL支持
- ✅ 日志和监控

**使用示例**：
```typescript
import { OpenAIAdapter } from './core/model/OpenAIAdapter';

const openaiAdapter = new OpenAIAdapter({
  apiKey: process.env.OPENAI_API_KEY!,
  organization: process.env.OPENAI_ORG_ID,
  baseUrl: 'https://api.openai.com/v1',
  model: 'gpt-4-turbo-preview',
  maxRetries: 3,
  timeout: 30000
});

await openaiAdapter.initialize();

const response = await openaiAdapter.chat({
  provider: 'openai',
  type: 'chat_llm',
  model: 'gpt-4-turbo-preview',
  messages: [
    { role: 'system', content: '你是一个有帮助的AI助手' },
    { role: 'user', content: '你好！' }
  ]
});

console.log(response.message.content);
```

### 4. ✅ 模型路由器（ModelRouter）

**路由策略**：
- 🔁 **轮询（Round Robin）**：平均分配请求
- ⚡ **最低延迟（Least Latency）**：选择响应最快的模型
- 💰 **成本优化（Cost Optimized）**：选择成本最低的模型
- 🎯 **质量优先（Quality First）**：选择质量最高的模型
- 🎛️ **手动指定（Manual）**：手动指定使用哪个模型

**核心功能**：
- ✅ 智能模型选择
- ✅ 多策略路由
- ✅ 自动故障转移
- ✅ 性能监控和统计
- ✅ 成本优化
- ✅ 健康检查
- ✅ 多模型管理

**使用示例**：
```typescript
import { ModelRouter, RoutingStrategy } from './core/model/ModelRouter';

const router = new ModelRouter({
  strategy: RoutingStrategy.QUALITY_FIRST,
  defaultProvider: 'anthropic',
  fallbackEnabled: true,
  routes: [
    {
      provider: 'anthropic',
      model: 'claude-3-opus-20240229',
      priority: 10,
      enabled: true,
      config: {
        apiKey: process.env.ANTHROPIC_API_KEY!
      }
    },
    {
      provider: 'openai',
      model: 'gpt-4-turbo-preview',
      priority: 9,
      enabled: true,
      config: {
        apiKey: process.env.OPENAI_API_KEY!
      }
    },
    {
      provider: 'local',
      model: 'llama3',
      priority: 1,
      enabled: true,
      config: {
        modelPath: '/models/llama-3-8b',
        engine: 'ollama',
        host: 'localhost',
        port: 11434
      }
    }
  ]
});

await router.initialize();

// 使用路由器
const response = await router.chat({
  provider: 'anthropic',  // 可选，如果不指定则使用路由策略
  type: 'chat_llm',
  model: 'claude-3-opus-20240229',
  messages: [
    { role: 'user', content: '你好！' }
  ]
});

console.log(response.message.content);

// 获取统计信息
const stats = router.getStats();
console.log('模型统计:', stats);

// 获取健康状态
const healthStatus = await router.getHealthStatus();
console.log('健康状态:', healthStatus);
```

## 🚀 快速开始

### 1. 配置环境变量

```bash
# .env
OPENAI_API_KEY=sk-your-openai-api-key
ANTHROPIC_API_KEY=sk-ant-your-anthropic-api-key
LOCAL_MODEL_PATH=/models/llama-3-8b
```

### 2. 创建路由器

```typescript
import { ModelRouter, RoutingStrategy, ModelProvider } from './core/model/ModelRouter';

const router = new ModelRouter({
  strategy: RoutingStrategy.QUALITY_FIRST,
  defaultProvider: ModelProvider.ANTHROPIC,
  fallbackEnabled: true,
  routes: [
    {
      provider: ModelProvider.ANTHROPIC,
      model: 'claude-3-opus-20240229',
      priority: 10,
      enabled: true,
      config: { apiKey: process.env.ANTHROPIC_API_KEY! }
    },
    {
      provider: ModelProvider.OPENAI,
      model: 'gpt-4-turbo-preview',
      priority: 9,
      enabled: true,
      config: { apiKey: process.env.OPENAI_API_KEY! }
    }
  ]
});

await router.initialize();
```

### 3. 发送请求

```typescript
const response = await router.chat({
```
