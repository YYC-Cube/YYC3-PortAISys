## 🚀 使用指南

### 快速开始

1. **环境配置**
   ```bash
   cp .env.example .env
   # 编辑 .env 文件，填入真实的API密钥和配置
   ```

2. **安装依赖**
   ```bash
   bun install
   ```

3. **运行项目**
   ```bash
   bun run dev
   ```

### 使用示例

#### 使用消息总线
```typescript
import { MessageBus } from './core/MessageBus';
import { MessageType } from './core/types/engine.types';

const messageBus = new MessageBus({
  maxQueueSize: 100,
  retryPolicy: {
    maxRetries: 3,
    backoffFactor: 2
  }
});

// 订阅消息
messageBus.subscribe(MessageType.USER_MESSAGE, async (message, context) => {
  console.log('收到消息:', message);
  return { success: true, content: 'Response' };
});

// 发布消息
await messageBus.publish({
  id: 'msg-001',
  type: MessageType.USER_MESSAGE,
  content: 'Hello AI',
  timestamp: new Date()
});
```

#### 使用自治AI引擎
```typescript
import { AutonomousAIEngine } from './core/AutonomousAIEngine';
import { MessageType } from './core/types/engine.types';

const engine = new AutonomousAIEngine({
  version: '1.0.0',
  environment: 'development',
  messageConfig: {
    maxQueueSize: 100,
    retryPolicy: { maxRetries: 3, backoffFactor: 2 }
  },
  taskConfig: {
    maxConcurrentTasks: 10,
    timeoutMs: 30000,
    priorityLevels: 4
  },
  stateConfig: {
    autoPersist: true,
    persistInterval: 60000,
    maxHistory: 1000
  },
  logConfig: {
    level: 'info',
    format: 'json'
  }
});

// 启动引擎
await engine.initialize({});
await engine.start();

// 处理消息
const response = await engine.processMessage({
  id: 'msg-001',
  type: MessageType.USER_MESSAGE,
  content: 'Hello AI',
  timestamp: new Date()
});

console.log('响应:', response);
```

#### 使用日志系统
```typescript
import { logger } from './core/utils/logger';

logger.debug('调试信息', 'Context', { key: 'value' });
logger.info('普通信息', 'Context');
logger.warn('警告信息', 'Context', { key: 'value' }, new Error('Warning'));
logger.error('错误信息', 'Context', { key: 'value' }, new Error('Error'));
logger.fatal('致命错误', 'Context', { key: 'value' }, new Error('Fatal'));
```

#### 使用指标收集
```typescript
import { metrics } from './core/utils/metrics';

// Counter：计数器
metrics.increment('api.requests', 1, { endpoint: '/chat', method: 'POST' });

// Gauge：仪表盘
metrics.gauge('memory.usage', 1024, { type: 'heap' });

// Histogram：直方图
metrics.histogram('api.latency', 150, { endpoint: '/chat', method: 'POST' });

// Summary：摘要
metrics.summary('response.size', 2048, { type: 'response' });

// 获取所有指标
const allMetrics = metrics.getAllMetrics();
console.log('所有指标:', allMetrics);
```
