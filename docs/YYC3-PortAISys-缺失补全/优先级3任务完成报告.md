## 📊 技术实现细节

### Worker系统架构

```
src/worker/
├── types.ts              # 类型定义
├── worker.ts             # Worker核心实现
├── worker-entry.ts       # Worker入口点
└── manager.ts           # Worker管理器
```

### 性能测试系统架构

```
src/performance/
├── types.ts              # 类型定义
├── engine.ts             # 性能测试引擎
└── suite.ts             # 性能测试套件
```

### Worker通信流程

1. **主线程** → 创建WorkerManager
2. **WorkerManager** → 初始化Worker池
3. **主线程** → 提交任务到WorkerManager
4. **WorkerManager** → 分配任务到空闲Worker
5. **Worker** → 执行任务并返回结果
6. **WorkerManager** → 收集结果并更新统计
7. **主线程** → 获取结果

### 性能测试流程

1. **初始化** → 创建PerformanceTestEngine
2. **基准测试** → 执行单线程和并发基准测试
3. **压力测试** → 逐步增加并发数，找到破坏点
4. **负载测试** → 在指定时间内维持目标RPS
5. **报告生成** → 汇总所有测试结果，生成报告
